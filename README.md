CV Generator

Prosjektbeskrivelse

Dette prosjektet er en nettside laget med Flask hvor brukere kan lage sin egen CV. 
Brukeren fyller inn informasjon som navn, e post, telefon, utdanning, arbeidserfaring og ferdigheter i et skjema. 
Når skjemaet sendes inn blir informasjonen lagret i en database og vist på nettsiden.

Prosjektet er laget som en tverrfaglig IT oppgave som dekker kompetansemål fra drift, brukerstøtte og utvikling.

Teknologier brukt
- Python
- Flask
- HTML
- CSS
- SQLite
- GitHub



Hvordan programmet fungerer

1. Brukeren åpner nettsiden.
2. Brukeren fyller inn informasjon i CV-skjemaet.
3. Informasjonen sendes til Flask-serveren.
4. Flask lagrer informasjonen i en SQLite database.
5. CV-en vises på nettsiden.
